# 🛒 Sistema de Vendas (PDV)

Sistema desktop desenvolvido em Java com JavaFX para controle de vendas, clientes e estoque.

---

## 🚀 Funcionalidades

- Cadastro de clientes
- Cadastro de produtos
- Controle de estoque
- Realização de vendas (PDV)
- Histórico de compras por cliente
- Baixa automática no estoque
- Métodos de pagamento (Dinheiro, Cartão, Pix)

---

## 🖥️ Telas do Sistema

### 🔐 Login
Permite acesso ao sistema conforme o tipo de usuário:
- Gerente
- Vendedor
- Estoquista

---

### 👥 Clientes
- Cadastro de cliente
- Validação de CPF e CNPJ
- Histórico de compras

---

### 📦 Produtos
- Cadastro de produtos
- Controle de quantidade em estoque

---

### 💰 PDV (Vendas)
- Adição de produtos no carrinho
- Remoção de itens
- Cálculo automático do total
- Cálculo de troco
- Finalização da venda

---

## ⚙️ Tecnologias Utilizadas

- Java
- JavaFX
- MySQL
- JDBC
